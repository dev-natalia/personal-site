# Portfólio — Natália Pereira da Silva

## Visão Geral

Site de portfólio pessoal de Natália Pereira da Silva, desenvolvedora backend sênior com +6 anos de experiência. O site tem como objetivo apresentar sua trajetória, skills, projetos e facilitar contato com recrutadores e empresas.

## Objetivo

Portfólio voltado para empresas mais conservadoras, com estilo "minimal fun" — limpo e organizado, mas com personalidade e identidade visual própria. Não é um site genérico.

## Stack

- **Framework:** Next.js (App Router)
- **Estilo:** Tailwind CSS
- **Animações:** Framer Motion
- **Ilustrações:** SVGs gerados no Recraft.ai, animados com Framer Motion
- **Integração:** GitHub API (projetos puxados dinamicamente)
- **Hospedagem:** A definir (Vercel recomendado)

## Estrutura do Site

One page com navegação por scroll suave (âncoras):

1. **Hero** — nome, tagline, ilustração animada, CTAs
2. **Sobre mim** — bio, trajetória, fatos pessoais
3. **Skills** — tecnologias organizadas por categoria, com logos
4. **Projetos** — cards com projetos próprios e integração GitHub
5. **Contato** — LinkedIn, GitHub, email e formulário

## Specs Detalhadas

- `specs/identity.md` — identidade visual completa
- `specs/sections.md` — conteúdo de cada seção
- `specs/projects.md` — detalhes dos projetos
- `specs/tech.md` — decisões técnicas e arquitetura
